RentalFlow is a Rental Property Management Web API developed using ASP.NET Core 8 following Clean Architecture principles to ensure scalability, maintainability, and separation of concerns. The application enables efficient management of rental properties through RESTful API operations such as creating, retrieving, updating, and deleting property records. The system is built using Entity Framework Core for data access and SQLite for database management, with dependency injection and repository pattern implementation to maintain a modular architecture. Swagger has been integrated for interactive API testing and documentation, allowing seamless verification of endpoints. The project demonstrates practical backend development concepts including layered architecture, database migrations, API design, and structured project organization suitable for real-world enterprise applications.
